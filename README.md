Deskripsi Program:
---> Project 1 - Parkir Mall Prienal
Program Parkir Mall Prienal merupakan aplikasi berbasis Command Line Interface (CLI) yang digunakan untuk membantu pengelolaan parkir mobil di Mall Prienal. Program ini dapat mencatat kendaraan yang masuk dan keluar, menghitung biaya parkir berdasarkan durasi parkir, menyimpan data petugas parkir, menampilkan daftar kendaraan yang sedang berada di area parkir, serta memberikan pemberitahuan ketika kapasitas parkir telah penuh. Program dikembangkan menggunakan konsep Class/Struct, Pointer, Array atau Vector, serta Function untuk menghasilkan sistem yang lebih terstruktur, mudah digunakan, dan efisien dalam mengelola data parkir.

---> Project 2 - Game Tebak Kata
Game Tebak Kata merupakan permainan berbasis Command Line Interface (CLI) yang mengharuskan pemain menebak kata rahasia huruf demi huruf sebelum nyawa habis. Program menyediakan beberapa tingkat kesulitan yang menentukan jumlah nyawa pemain, memilih kata secara acak dari bank kata yang tersedia, serta menampilkan perkembangan permainan secara interaktif. Selain itu, program dilengkapi validasi input, penyimpanan riwayat tebakan salah, sistem menang dan kalah, serta opsi untuk bermain kembali setelah permainan selesai.

---> Project 3 - Sistem Manajemen Inventaris Toko
Program Sistem Manajemen Inventaris Toko merupakan aplikasi berbasis Command Line Interface (CLI) yang digunakan untuk mengelola data barang pada sebuah toko retail. Setiap barang disimpan menggunakan struktur data yang berisi kode barang, nama barang, harga, dan stok. Program menyediakan fitur untuk menambah barang, menampilkan seluruh data barang, mencari barang dengan harga tertinggi, menghitung total nilai inventaris, mengecek barang yang perlu direstock, serta memperbarui stok barang menggunakan pointer. Program juga dilengkapi validasi input untuk memastikan harga dan stok tidak bernilai negatif.

---> Project 4 - Minesweeper++
Minesweeper++ merupakan permainan berbasis Command Line Interface (CLI) yang terinspirasi dari game Minesweeper klasik. Pemain dapat menentukan ukuran papan permainan dan jumlah bom sebelum permainan dimulai. Tujuan permainan adalah menemukan seluruh bom dengan menandainya secara tepat tanpa membuka kotak yang berisi bom. Program dilengkapi dengan fitur pembukaan area kosong secara otomatis (flood fill), sistem penandaan bom, penghitung waktu permainan, serta kondisi menang dan kalah yang interaktif. Seluruh logika permainan dikelola menggunakan satu class utama yang menangani papan permainan, posisi bom, status kotak, dan mekanisme permainan secara keseluruhan.

---> Project 5 - Game Tebak Negara
Game Tebak Negara merupakan permainan berbasis Command Line Interface (CLI) yang menguji pengetahuan pemain mengenai berbagai negara di dunia. Pemain akan diberikan petunjuk tertentu dan diminta menebak nama negara yang benar. Program dirancang agar interaktif dan mudah dimainkan melalui terminal, dengan mekanisme penilaian berdasarkan jawaban yang diberikan pemain. Selain sebagai hiburan, permainan ini juga dapat membantu menambah wawasan geografis dan pengetahuan umum mengenai berbagai negara.
